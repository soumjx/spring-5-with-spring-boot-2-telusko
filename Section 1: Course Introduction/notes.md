# 2 - Projects in this course

## Course Structure
- Core Java → Maven, Hibernate → Spring Framework → Docker Microservices → Spring AI

## Project 1: Quiz Application (Core Java)
- Simple quiz app testing Java syntax knowledge
- Asks questions (e.g., "size of int"), calculates score
- 99% code written line-by-line in course
- Same app rebuilt later with microservices

## Project 2: Job Portal (Spring Framework)
- Backend: Job service, repository classes
- Features: Create jobs (ID, experience, description, skills), delete jobs
- Frontend provided (not taught) to visualize backend data
- Full CRUD operations between UI and backend

## Project 3: E-commerce Application (Spring REST + Spring AI)
- Features: Products management, search, cart, checkout, orders
- Two versions: Without AI and with AI capabilities
- **AI Features:**
  - AI-generated product descriptions
  - AI-generated product images
  - Chatbot for order status (email-based lookup)
- Docker integration for database
- Search by brand/color, LLM integration (OpenAI/other providers)

## Project 4: Quiz Microservices
- Same quiz app rebuilt with microservices architecture
- Services: API Gateway, Question Service, Quiz Service
- Multiple small services communicating via API Gateway

## Key Points
- Write 99% code line-by-line (some config/AI suggestions copied)
- Backend-focused (frontend provided for visualization)
- Practice every video by coding yourself
- Team support for questions
